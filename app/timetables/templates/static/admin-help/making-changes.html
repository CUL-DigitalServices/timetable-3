    Making changes

    Why would you want to edit events?

    If a lecturer is off sick and his teaching assistant will instead take the lecture, you may want to update the name in the timetable, or if there are building works going on in a particular room, you may want to update the location for a couple of seminars. This guide will give you an overview on how to do that.

    Editing events - a run through

    Timetables overview page: selecting relevant departments and viewing editable and non-editable timetables

    The Timetables overview page allows you to select a department or tripos, as well as giving an overview of the different timetables associated with that department or tripos. In most cases, you'll just want to look at the department you are responsible for, but you could have a look at other departments as well if you wished to do so.

    A pencil icon indicates the timetables you have edit rights to, the others are view only.

    When being taken to a timetable on which you have edit rights, the most straightforward way to edit events is by looking at the list view. In the future it should be easier to make these changes in the week view as well.

     

    Editing one event

    The list view shows an overview of all the modules within that timetable, and lecture series containing all the individual events for that series underneath it. You can click on the expanding arrow next to a lecture series title to see the list of individual events. Each row shows another event with the relevant data associated with that event: event title - event type - date/time - location/room - lecturer/presenter.

    On the right side of each event, you will see a couple of icons, and the pencil icon (i.e. edit) is one of those. When hovering over the event, you will see that the event fields become yellow and pencils show up next to each field. Clicking any of these pencils will turn the fields into edit mode allowing you to start editing. As soon as you change anything, a save button appears.

    Editing the event title, location or lecturer
    Clicking a pencil icon, or on the text itself, allows you to make text edits inline - just click on the text and make your changes.

    Editing the event type
    When clicking the type drop down, different options appear which allow you to select the relevant type of event.

    Editing the event date/time
    When clicking the date/time field, a small pop up shows up with different fields. Timetable uses the Cambridge University calendar, so the weeks relate to the Cambridge weeks with weeks starting on Thursdays. Entering a negative week, e.g. '-2', allows you to add events which take place before the beginning of the academic calendar. The time fields give you a set of drop downs, but allow typing if you'd prefer to enter the data that way. If you want to change the times, you can use the 'from' and 'to' field. These won't allow you for the 'to' and 'from' field to be the same, so you'll notice you'll have to change the 'to' field first if you want the lecture to start at the time it originally ended. The 'to' field will automatically change if you change the 'from' field. Clicking the 'okay' button will close the pop up so you can continue editing.

    When you've made changes, you will see a save button underneath the list of events. When clicking the save button, your changes will be saved, and these updates will be visible to students. You will see a confirmation that your changes have been saved. Clicking the okay button will take you back to the timetable list view.

    Editing multiple events

    If you have to change the location for three lectures within one lecture series, you can edit each of them and then click the save button once. It is possible to make these updates to different fields at the same time if you want to.

    At the moment, you can also edit multiple events across various series or modules. Please ensure you have clicked a save button for each group you have made changes to.

    Editing a lecture series or module title

    To change a module or series title, click on the pencil which you see when hovering over it.

    The title will become yellow which means it's editable and you can just start editing it. Clicking outside the editable field saves this change, and will automatically update in the student's view.

    You can cancel and edit simultaneously

    You don't have to treat editing and cancelling as two separate tasks. For example, if you want to change the location of a seminar, but at the same time you also want to cancel another event, you can do this all at once before hitting the save button. If you want to find out how to cancel events, have a look at the cancel guide.
