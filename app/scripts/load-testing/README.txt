=======================
Timetables Load Testing
=======================

This directory contains files related to load testing Timetables.

create_test_users.py is a script to create an arbitrary number of test users on
the test system

The "tsung" directory contains a Tsung test plan. This is the primary tool for
load-testing Timetable.

The "jmeter" directory contains a JMeter test plan. This has been deprecated.
